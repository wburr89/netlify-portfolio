Projects

Covid Tracker

This is a Covid-19 information tracker with data provided by John Hopkins University. Users can search counties in the U.S to get covid data. Users can see overall data concerning the effects of the virus on the U.S population. There is a Vaccine trial page listing the top players who are developing the vaccine, those funding it, and a brief description of the trial. I created this web application with: HTML, CSS, javascript, Jquery, Bootstrap and leaflet.JS

Text-To-Speech App

This is a Vanilla Javascript heavy application using the built in SpeechSynthesisUtterance() method that was built with children accessiblity in mind. Children who have trouble speaking can click on the image that resembles what they are feeling at the time. It is also possible for the user to input custom text that will be read aloud.
